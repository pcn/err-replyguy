# errbot-weather

This is a plugin for errbot which stores responses to questions. 

## install
- setup & use virtualenv
- pip install -r requirements.txt

## usage
- see !help
- !rg add <trigger string> <reply string>
- !rg del <trigger string>
- !rg <trigger string>
- !rg list


## example

```
!rg reply guy
```
